# Practical Programming with C: Daily Lab Problems

Welcome to my repository for the "Practical Programming with C" lab sessions. This project serves as a comprehensive collection of my daily lab work and problem-solving exercises in the C programming language.

The purpose of this repository is to:

- Track daily progress and a history of my solutions.
- Store and organize C code for easy reference.
- Demonstrate my understanding of fundamental programming concepts in C.

## Repository Structure

The problems are organized in a clear, consistent manner. Each problem solution is placed in its own folder or file, with a descriptive name.

## About the Subject

Practical Programming with C is a core subject focused on building a strong foundation in procedural programming. Key topics covered include:

- Basic syntax and data types
- Control flow (if-else, switch, loops)
- Functions and modular programming
- Arrays, strings, and pointers
- File I/O
- Data structures (as needed)

## How to View the Code

You can browse the folders directly in this repository. To run any of the C files on your local machine, you will need a C compiler (such as GCC).

To compile and run a C file:

```sh
# Navigate to the directory containing the file
cd path/to/your/file

# Compile the code (replace 'filename' with the actual file name)
gcc filename.c -o output_name

# Run the compiled program
./output_name
```
